Here's a review of your SQL statement:

### Original Code
```sql
select from Customer where CustomerID>30
```

### Issues

1. **Missing SELECT columns**:  
   The statement does not specify what columns to select. In SQL, you must either specify the column names or use `*` to select all columns.

2. **Optional but recommended:**  
   SQL keywords are often in uppercase for readability (`SELECT`, `FROM`, etc.).

### Corrected Version

To select all columns:
```sql
SELECT * FROM Customer WHERE CustomerID > 30;
```

Or, to select specific columns (e.g., `CustomerID`, `Name`):
```sql
SELECT CustomerID, Name FROM Customer WHERE CustomerID > 30;
```

---

**Summary:**  
- Always specify columns after `SELECT` (or use `*` for all columns).  
- Capitalize SQL keywords for standard convention.
